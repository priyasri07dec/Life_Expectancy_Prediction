# Life_Expectancy_Prediction

A Machine Learning web application built using Python and Streamlit that predicts the life expectancy of a country based on various health, economic and social factors.

### Project Overview

This project uses Machine Learning techniques to analyze factors affecting life expectancy and predict the expected lifespan using input features such as GDP, healthcare indicators, immunization rates, schooling and mor.
The application provides an interactive user interface where users can enter values and get instant predictions.

### Features

* Interactive streamlit web application
* Data preprocessing and feature scaling
* Machine Learning model for prediction
* Clean and user friendly UI
* Model deployment ready

### Technologies Used

* Python
* Pandas
* Numpy
* Scikit-Learn
* Streamlit
* Joblib
* Pickle

### Dataset

The dataset contains country wise information related to:
* GDP
* Schooling
* Adult Mortality
* BMI
* Alcohol Consumption
* Immunization Coverage
* Healthcare Expenditure
* Population
* Income Composition
* and other socio economic indicators

### Machine Learning Workflow

1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis(EDA)
4. Feature Engineering
5. Feature Scaling
6. Model Training
7. Model Evaluation
8. Deployment using Streamlit
